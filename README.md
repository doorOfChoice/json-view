# json-view



## INSTALL

```
go get github.com/doorOfChoice/json-view
go build -o json
./json [file]
```

## DEPENDENCE

``github.com/nsf/termbox-go``

## OPERATION

```
[↑] move up
[↓] move down
[←] move left
[→] move right
[F2] close toggle
[F3] open all json
[W] move up 5 unit
[S] move down 5 unit
[Ctrl + C] exit 
```
